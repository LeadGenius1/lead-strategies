# How to Send Email to Pinnacle PCS

## Quick Send Instructions

### Method 1: Copy & Paste (Easiest)
1. Open `email-pinnaclepcs-ready-to-send.txt`
2. Copy all content
3. Open your email client (Gmail, Outlook, etc.)
4. Create new email
5. **To:** info@pinnaclepcs.com
6. **Subject:** Help Pinnacle PCS Win More Commercial Cleaning Contracts in Berks County
7. Paste the email body
8. Send!

### Method 2: Use Platform Dashboard
1. Login to https://leadstrategies-frontend.vercel.app/login
2. Navigate to Dashboard → Campaigns
3. Click "+ New Campaign"
4. Use the email content from `email-pinnaclepcs-ready-to-send.txt`
5. Set recipient: info@pinnaclepcs.com
6. Send campaign

### Method 3: Via Backend API (Requires Auth)
The backend has `/api/campaigns/send` endpoint but requires authentication token.

To use:
1. Login to get auth token
2. Use the token in API request
3. Send POST to: https://backend-production-2987.up.railway.app/api/campaigns/send
