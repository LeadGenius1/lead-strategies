# ClientContact.IO Frontend (Tier 3)

Multi-channel outreach platform frontend for ClientContact.IO.

## Features

- Multi-channel management (22+ platforms)
- Unified inbox
- AI-powered content generation
- Real-time analytics
- Team collaboration tools

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Plus Jakarta Sans fonts
- AOS animations
- Typed.js

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_API_URL=https://api.leadsite.ai
NEXT_PUBLIC_API_VERSION=v1
```

## Deployment

Deploy to Vercel for `clientcontact.io` domain.
