# Cursor IDE Capabilities Reference

## Purpose
Reference for Cursor IDE integration with AI Lead Strategies Platform.

## Core Capabilities
- Multi-file editing and code generation
- Git operations (commit, push, branch)
- Terminal and shell execution
- File search and navigation
- Composer for complex multi-step tasks
- Rules and project context (.cursor/rules)

## Project Integration
- Source of truth: lead-strategies-repo
- Deployment: Railway (auto on push to main)
- Rules: source-of-truth.mdc, deployment-workflow.mdc
