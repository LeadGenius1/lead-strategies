### Custom Domain Setup

**Add your domain:**

1. Go to "**Settings**" ‚Üí "**Domains**"
2. Click "**+ Add Domain**"
3. Enter your domain (example.com)
4. Follow DNS setup instructions:
   - Add A record pointing to our IP
   - Add CNAME for www subdomain
5. Wait for verification (5-10 minutes)
6. SSL certificate auto-generated

**DNS Records:**
```
Type: A
Name: @
Value: [Our IP address shown in dashboard]

Type: CNAME
Name: www
Value: leadsite.io
```

## üìù Best Practices

### Website Design Tips

**Hero Section:**
- Clear, benefit-focused headline
- Specific subheadline
- Strong CTA ("Get Started Free", "Book Demo")
- Professional hero image or gradient

**Features:**
- Focus on benefits, not just features
- Use icons for visual appeal
- Keep descriptions concise (1-2 sentences)
- Limit to 3-6 features (don't overwhelm)

**Testimonials:**
- Use real customer quotes
- Include full names and companies
- Add photos for credibility
- Show variety of customers

**Pricing:**
- Show 2-3 plans max on landing page
- Highlight recommended plan
- Be transparent about pricing
- Include clear feature comparisons

**Contact Form:**
- Keep fields minimal (name, email, message)
- Clear submission button
- Thank you message after submit
- Fast response time promise

### Conversion Optimization

**Above the Fold:**
- Clear value proposition
- One primary CTA
- Visual hierarchy
- Professional imagery

**Social Proof:**
- Customer logos
- Testimonials
- Case studies
- Trust badges

**Mobile First:**
- Test on mobile devices
- Ensure text readable
- Buttons easy to tap
- Fast loading time

## üîß Troubleshooting

### Builder not loading

**Solutions:**
- Refresh page
- Clear browser cache
- Try different browser
- Check internet connection

### Can't publish website

**Check:**
- Website has at least one section
- All required fields filled
- No JavaScript errors
- You're logged in

### Custom domain not working

**Verify:**
- DNS records added correctly
- Allow 24-48 hours for propagation
- SSL certificate generated (automatic)
- No typos in domain name

### Images not displaying

**Check:**
- Image URL is correct and accessible
- Image file size < 5MB
- Use supported formats (JPG, PNG, WebP, GIF)
- Try different image host
