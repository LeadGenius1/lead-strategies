- `/hours` - "Our business hours are..."

**Usage:** Type `/` in any conversation to see templates

## ğŸ¤– Auto-Responses

### Create Automation Rules

1. Go to "**Automation**" tab
2. Click "**New Rule**"
3. Configure rule:
   - **Name:** "After hours auto-reply"
   - **Trigger:** "Time-based"
   - **Condition:** "Outside business hours"
   - **Channel:** "All channels" or specific
   - **Response:** "Thanks for your message! We'll reply during business hours..."
   - **Priority:** High (1-10)
   - **Delay:** 0 seconds (or set delay)

4. Toggle "**Enabled**"
5. Click "**Save Rule**"

### Automation Trigger Types

**1. Time-Based**
- Outside business hours
- Weekends
- Holidays
- Specific time ranges

**2. Keyword-Based**
- Contains specific words
- Customer asks about pricing
- Customer says "urgent"

**3. Channel-Based**
- First message on SMS
- First email from customer
- Chat widget initiated

**4. Status-Based**
- New conversation
- Conversation reopened
- No response in X hours

## ğŸ†˜ Troubleshooting

### Messages not syncing

**Check:**
- Channel connected correctly
- API credentials valid
- Webhook endpoint responding
- Internet connection stable

**Solution:**
- Reconnect channel
- Verify credentials
- Check webhook logs
- Restart sync

### Canned responses not inserting

**Check:**
- Shortcode spelled correctly (case-sensitive)
- Template is enabled
- You're in reply field

**Solution:**
- Type `/` to see available templates
- Copy-paste if shortcode not working
- Refresh page

### Auto-response not triggering

**Check:**
- Rule is enabled
- Trigger conditions met
- No conflicting rules
- Channel is correct

**Solution:**
- Review rule configuration
- Test with manual trigger
- Check rule logs
- Verify priority settings
