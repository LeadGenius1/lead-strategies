## ðŸ”„ Sales Sequences

### Create Automated Outreach

**What are sequences?**
- Multi-step automated email campaigns
- Timed follow-ups
- Task creation
- Smart stopping rules

**Create Sequence:**

1. Go to "**Sequences**"
2. Click "**+ New Sequence**"
3. Add steps:
   - **Step 1:** Initial email (Day 0)
   - **Step 2:** Follow-up email (Day 3)
   - **Step 3:** LinkedIn connection (Day 7)
   - **Step 4:** Phone call task (Day 10)
   - **Step 5:** Final email (Day 14)

4. Set rules:
   - Stop if contact replies
   - Stop if deal created
   - Skip weekends
   - Skip holidays

5. Click "**Save Sequence**"

**Enroll Contacts:**
1. Open sequence
2. Click "**Enroll Contacts**"
3. Select contacts
4. Click "**Start Sequence**"

**Monitoring:**
- View enrollment status
- Track open rates
- Track reply rates
- See who's in each step

## ðŸ”§ Troubleshooting

### Deal not appearing in pipeline

**Check:**
- Deal has pipeline assigned
- Deal is not archived
- You have permission to view
- Filters not hiding deal

### Can't create company

**Check:**
- Company name not duplicate
- Domain format correct (example.com)
- Required fields filled
- You have permission

### Bulk import failed

**Common issues:**
- CSV format incorrect (check headers)
- Required fields missing
- Duplicate entries
- Invalid email addresses

**Solution:**
- Download CSV template
- Match column headers exactly
- Remove duplicates
- Validate emails
